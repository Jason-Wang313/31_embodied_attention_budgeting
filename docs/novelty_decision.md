# Novelty Decision

Chosen thesis:

Treat embodied attention as a limited physical resource that must be budgeted against control risk.

Why this thesis wins:

- It directly attacks a common field assumption.
- It changes the central mechanism from "pick informative observations" to "ration sensing under risk."
- It is not satisfied by generic active perception, generic uncertainty estimation, or generic sensor scheduling.
- It can be tested in a small runnable environment.

Rejected directions:

- bigger model,
- better data,
- new benchmark only,
- add uncertainty,
- add active learning,
- add verifier,
- combine two existing modules,
- use an LLM planner,
- use RL as the main novelty.

Paper form:

- a mechanism paper with a toy but adversarially designed embodied control task,
- plus a robotics-facing argument and evidence that the assumption matters.

Readiness:

- promising if the toy evidence and literature boundary are sharp,
- otherwise workshop-level.
